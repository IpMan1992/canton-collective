# CC Research Protocol — Social-Velocity-First

## Signal hierarchy (inverse of HAE — this customer is discovery-led)

1. **Social velocity (highest weight)** — TikTok, Instagram, Pinterest, Reddit. What's accelerating in GenZ/Y2K/alt/coquette/grunge communities right now. Look for styling and "outfit" content and rising hashtags/sounds — distinguish genuine adoption from one-off viral spikes.
2. **Aesthetic-community & search momentum** — named -cores and their wearable residue, Pinterest Predicts / Pinterest trend reporting, Google/TikTok search breakouts, Depop/Grailed resale heat for alt pieces.
3. **Loose comps (confirmation only)** — Dolls Kill, I AM GIA, indie/Asian-forward labels: "are they dropping/pushing this too?" Never primary.
4. **Asian street/social scenes (early signal + home turf)** — C-fashion, J-fashion, K-fashion; Weibo/Xiaohongshu (RED)/Japanese street-style reporting where accessible. Doubly relevant given AU + Japan customers.

Trade forecasters (WGSN/Vogue Business/BoF) are CONTEXT only here — they lag this customer.

## Search strategy

Always include current month/year. Patterns (adapt, don't recite):
- "TikTok fashion trends [month year]" / "[aesthetic]core trend [year]" (coquette, grunge, Y2K, mall goth, blokette, etc.)
- "Pinterest [year] trend report fashion" / "Pinterest predicts [year]"
- "Gen Z fashion trend [month year]" / "Depop trending [season year]"
- "[specific microtrend] trend [month year]" — bows, leg warmers, fishnets, baby tees, cargo, lace, charm jewelry, polka dots, etc.
- "Reddit r/AltFashion OR r/Y2KFashion trending [year]" style queries
- "K-fashion / J-fashion trend [season year]"
- comp checks: "Dolls Kill new [season]", "I AM GIA new arrivals"

Deep Dive: 12–18 searches across all dimensions + 4 sub-brands. Pulse: 6–8, focus on deltas + push-now.

## Lifecycle placement (drives the action lane)

- **Rising** — signal accelerating, not yet saturated, comps not all-in → POSITION EARLY (runway to feature/buy before crest).
- **Peaking / near-peak** — saturated across platforms, comps moving, search/resale hot NOW → PUSH NOW if catalog matches exist (no lead-time risk on owned stock).
- **Fading** — declining velocity, comps discounting, "is X over?" content appearing → CLEAR ONLY.

Microtrends move fast: re-place every holdover trend each run; a Push Now from last ledger may be Clear Only now. State the evidence for the placement, don't assert it.

## Evidence-block standard (team-grade — the differentiator)

Every recommended trend must be defensible to a buyer/marketer without re-research:
- **Signal** — named platform(s) + specific observable, dated.
- **Driver** — why now (cultural moment, celebrity, season, show, sound) if identifiable.
- **Lifecycle + runway** — placement + estimated remaining window, with supporting signal.
- **CC opportunity & proof** — matching SKU count + names + sub-brand(s); whether any are ALREADY in recent sales climbers/winners (cite the number); the specific action.
- **Sourcing note** (Position Early) — what to raise with the sub-brand for the next drop.

## Confidence

- **High** — social velocity + aesthetic-community + comp/resale confirmation agree, AND catalog matches exist (ideally already climbing).
- **Med** — strong single-platform signal or signals agree but thin catalog match.
- **Low** — single viral source or conflicting signals → cap at Position Early/Watch, never Push Now.

## Mapping to the ledger's status icons

The shared ledger (`memory/ledgers/trend.md`) uses its own momentum vocabulary — translate at write time:
- **▲ Accelerating** — signal rising vs last run (or strong rising new signal).
- **▶ Steady** — holding since last run.
- **▼ Cooling** — decelerating vs last run → usually CLEAR ONLY in the Directive.
- **🆕 New** — not present in the prior run block.
The lane (PUSH NOW / POSITION EARLY / CLEAR ONLY) goes in the Directive column; the icon captures momentum. They're related but not identical — a Steady ▶ trend with fresh catalog depth can still be PUSH NOW.

## Discipline

- Separate genuine adoption (styling/wear content, repeat creators) from algorithmic one-hit virality.
- A trend already declining on TikTok may still convert briefly for CC — mark Clear Only, don't lean in.
- Always tie back to catalog + sales. External signal alone never produces a Push Now.
- Note when matching is weak because products lack aesthetic tags — recommend tagging as infrastructure.
